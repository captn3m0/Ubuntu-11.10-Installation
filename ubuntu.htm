Installation

Partitioning

Read the article on partitions.
One can have a maximum of four partitions on a hard disk. If more are to be made, a pseudo partition (called extended partition) which can contain any number of partitions inside, has to be made. In our installations 2 additional partitions will be made. One will be for swap which is similar to pagefile.sys in windows and the other will be root.

Booting & Installation

  1. Download Unetbootin from here. Unetbootin is only required if your computer cannot boot from a CD. Otherwise, get a Ubuntu ISO, burn it (or ask for a disk from someone) and boot with it
  2. You'll need an empty usb drive which shall now be created into a bootable drive. Choose "Diskimage" and select the .iso file of Ubuntu 11.10. Create the drive.
  3. Insert the usb drive and reboot. Press F9 or F12 (depending on manufacturer) to change boot order into usb.
  4. Select default option when blue list loads and wait for Ubuntu to boot.
  5. Open "install ubuntu" shortcut and press continue till you reach "Installation type". Now select "something else".
  6. Select the partition that you predecided for swap and click on "Change". Select "Use as" option as "swap area". Click on ok.
  7. Now select the partition that you want to make the root partiton and click on "Change". Now select "Ext4 Journalling file system" for "Use as". Tick format the partition. Select "/" as your mount point. This will be your root partition.

  8. Optional

  9. You may also want to auto-mount your ntfs partitions (C and D drives) at boot. To do this select one of your ntfs partitions, click on "Change", set "Use as" to "ntfs" Warning don't tick on "Format the partition" option otherwise your data will be roast!
  10. Type "/media/c" (without quotes) in "Mount point". C is your drive here. You may replace c by anything you want but don't keep same mount point for different partitons eg c and d drive. For instance, you may mount your drives as /media/OS and /media/Entertainment
  11. Do this for your other ntfs partitions too. Now you are ready to install Ubuntu!
  12. Click on install now. You will be asked for some information (like Time Zone, user accounts etc). Next time it boots, you'll have your new Ubuntu ready to ride!
